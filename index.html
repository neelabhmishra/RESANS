RESANS

REST services for analytics as a service

Download this project as a .zip file Download this project as a tar.gz file

RESANS (RESt ANalytics Services) is a REST API that provides data analysis and predictive modeling services as RESTful web services.

The [init] version will allow calls for performing standard time-series forecasting analysis on continuous data. Analysis API's will be data and platform independent.
